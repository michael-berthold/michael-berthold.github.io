# Michael Berthold - Personal Website

A simple Jekyll-based personal website showcasing my work, publications, and musings.

## Structure

- `/about/` - About me page (available in English and German)
- `/musings/` - Blog posts and articles
- `/videos/` - Presentation videos
- `/publications/` - Technical articles, books, and academic publications
- `/_posts/` - Jekyll blog post content

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` to view the site locally.
